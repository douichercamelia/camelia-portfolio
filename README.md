## Lancer le portfolio

### Solution 1 — Via VS Code (recommandé)

1. Télécharger l'extension **Live Server** sur VS Code
2. Ouvrir le dossier du portfolio
3. Faire un clic droit sur index.html
4. Cliquer sur **Open with Live Server**

Le site s'ouvrira automatiquement dans le navigateur.

### Solution 2 — Via GitHub Pages

Le portfolio est également accessible directement en ligne :

https://douichercamelia.github.io/camelia-portfolio/
